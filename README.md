# Cyber Shield

Cyber Shield is a cybersecurity awareness platform designed to help organizations improve employees' security behavior through awareness training and simulated cyber attacks.

The platform connects administrators and employees through a role-based system. Administrators can create and manage awareness campaigns, including phishing simulations. Employees participate in these campaigns, access security awareness resources, complete quizzes, and report suspicious messages.

The system analyzes employee interactions and provides a Security Awareness Level score to help measure and improve cybersecurity awareness.

## User Roles

### Admin
- Create and manage awareness campaigns.
- Send phishing simulation campaigns to employees.
- Monitor campaign results and employee responses.
- View reports and analytics.

### Employee
- Receive and interact with awareness campaigns.
- Access cybersecurity training materials.
- Complete awareness quizzes.
- Report suspicious messages.
- View their Security Awareness Level.

## Features
- Role-based authentication.
- Phishing and security awareness simulations.
- Employee awareness scoring.
- Suspicious message reporting.
- Training resources and quizzes.
- Admin dashboard and reports.

## Technologies
- HTML
- CSS
- JavaScript
- PHP
- MySQL


## Project Purpose
The platform aims to enhance cybersecurity awareness and help organizations measure and improve employee security behavior.
